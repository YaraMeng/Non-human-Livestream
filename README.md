# Non-human-Livestream

这是一个面向“非人类直播”主题的项目仓库，用于整理代码、AI 生成工作流、过程文档和演示材料。

## 项目结构

- `code/`：所有代码、脚本和配置文件
- `workflow/`：视频和 AI 生成工作流，包括 ComfyUI 配置、提示词和执行步骤
- `docs/`：文档、反思和过程记录
- `presentation/`：演示 PPT 和导出的 PDF
- `assets/`：图片、音频、模型和其他素材
- `.gitignore`：忽略本地环境和生成文件


## 运行方法

1. 先确认 `code/` 中的脚本或项目依赖。
2. 如有 Python 项目，通常先创建虚拟环境并安装依赖。
3. 如有前端或 Node 项目，通常先安装依赖再启动本地服务。
4. 生成类任务优先查看 `workflow/` 中的提示词和步骤说明。

## 建议的工作方式

1. 先在 `docs/` 记录目标、进展和问题。
2. 生成素材统一放入 `assets/` 或其子目录。
3. 代码和配置集中放在 `code/`，避免散落在仓库根目录。
